# Image Placeholders

Thư mục này chứa hình ảnh cho website.

## Cấu trúc:

```
images/
├── games/           # Logo các game
│   ├── roblox.jpg
│   ├── lienquan.jpg
│   ├── freefire.jpg
│   └── genshin.jpg
├── products/        # Hình sản phẩm
│   ├── robux400.jpg
│   ├── robux800.jpg
│   └── ...
└── banners/         # Banner trang chủ
    └── hero.jpg
```

## Ghi chú:

Hiện tại website sử dụng emoji làm placeholder. Bạn có thể:

1. Thêm ảnh thật vào các thư mục tương ứng
2. Cập nhật đường dẫn trong database
3. Hoặc giữ nguyên emoji cho giao diện vui nhộn

## Kích thước đề xuất:

- **Game logos**: 200x200px (vuông)
- **Product images**: 400x400px (vuông)
- **Banners**: 1200x400px (ngang)

## Format:

- JPG/PNG cho ảnh thường
- WebP để tối ưu tốc độ
